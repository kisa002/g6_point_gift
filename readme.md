# g6_point_gift
## 그누보드6 포인트 선물 플러그인
플러그인 적용으로 간단하게 유저간 포인트를 선물할 수 있습니다.</br>
`관리자 대시보드` 내 `포인트 선물 내역`에서 `사용자들의 선물 내역`을 확인할 수 있습니다.

## 사용방법
### 다운로드
#### A. Git in terminal
1. cd {gn6_project}/plugin
2. git clone https://github.com/kisa002/g6_point_gift.git

#### B. 수동 다운로드
1. 그누보드6 프로젝트 최상단에 있는 plugin 폴더에 본 프로젝트를 압축해제 합니다.

### 플러그인 적용
1. `관리자 대시보드` 내 `환경설정` - `플러그인 관리`에서 `포인트 기프트` 항목에 있는 `플러그인 적용`을 클릭합니다.
2. 사이트 주소 뒤에 /bbs/point_gift에 접속하면 포인트를 선물할 수 있습니다.</br>
   ex) http://SERVER_ADDRESS/point_gift
